# Проект: Место

## О сайте
Интерактивный сайт с функцией добавления и удаления своих карточек, а также смены имени и описания профиля.

* **Grid-layout**
* **Flex-box**
* **Всплывающее окно**
* **Позиционирование**
* **Адаптивная вёрстка**
* **Javascript**
